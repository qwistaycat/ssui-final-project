# ssui-final-project
This is a project inspired by [Flexbox Froggy](https://flexboxfroggy.com/) that teaches some basic affine transformations with matrix multiplication!
### The website is deployed here: https://ssui-final-project.vercel.app/level/1

## Instructions for running locally:
```
cd my-app
npm install
npm run dev
```
